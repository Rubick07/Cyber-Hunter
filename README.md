## 🔴About
Cyber Hunters is a 2D Turn Based Strategy RPG in which you play as Edward, a young Cyber Hunter accompanied with Faye a lab assistant partnered with him to dive and protect the cyber world. The game is set on 2099, where technology has advanced further beyond and humans are now capable of diving into the virtual world.

Enter the virtual world in the future to uncover the secrets of phishing. Taking the perspective of a young cyber hunter tasked with exploring the virtual world to find clues and solve puzzles, while learning the concepts of cybercrime phishing to catch the perpetrators along with his partner, a genius lab assistant.

##  📜Scripts and Features

- In this game, we need to clear the stage before certain turn.
- in some Nodes there is some obstacle that need to clear with playing minigames.

|  Script       | Description                                                  |
| ------------------- | ------------------------------------------------------------ |
| `Node.cs` | Manages Player move, enemy pathfinding to search player|
| Laser.cs | Manages Laser for Light minigames |
| `GameManager.cs`  | Manages the game state, Enemy Move and turn |
| `LevelUnlock.cs`  | Manages levels |
| `Buff.cs`  | Manages buff for player |
| `etc`  | |

      ├── Image                   # In this folder, it contaions all sprites, background, UI image, etc
      ├── Prefab                   # In this folder, it contaions preafab
      ├── Scenes                     # In this folder, there are scenes. You can open these scenes to play the game via Unity
      ├── Scripts                     # In this folder, it contaions all script for the games
      ├── SO                         # In this folder, it contaions all Scriptable Object
      ├── Suara                      # In this folder, it contaions all SFX and BGM
      ├── TextMeshPro                 # In this folder, it contaions plugin for TextMeshPro
      ├── ....
   ├── ...
